# Pong Game

Pong is a simple two-player game where each player controls a paddle to hit the ball.

## How to Build
Run the following command to build the game:
```
cargo build --release
```

## How to Run
Execute the game with:
```
cargo run
```

## Controls
- Player 1: W (up) and S (down)
- Player 2: Up/Down arrow keys
- Q to quit

## Game Rules
The first player to reach 10 points wins.
